Description

Given a linked list, remove the nth node from the end of list and return its head.

For example,

Note:
Given n will always be valid.
Try to do this in one pass.

Solution

Analysis

  1. In order to find this node in one pass, we maintain two pointers and then move the current pointer forward with n steps.
  2. Keep this interval and move two pointers at the same speed until the current pointer reaches to the end.
  3. The previous one points to the previous node of target, then delete this target.
  4. The most important thing is we need ensure what we changed is in the same linked list.
