本篇关于Leetcode-Copy List with Random Pointer的算法.

Description:

A linked list is given such that each node contains an additional random pointer which could point to any node in the list or null.

Return a deep copy of the list.

Solution:

Analysis:

This is my wrong solution. Although this solution can pass the test cases, every next and random node only has label but no its next and random node.

The correct solution is we need use the new existed node as a next and random node.

First solution:

  1. we utilize HashMap to collect all copy node only with the value;
  2. Each time, we add next and random pointer which is only in this HashMap.
  3. Time complexity and space complexity is O(n)

Second solution:

  1. Iterate the original list and duplicate each node. The duplicated node follows its original node.
  2. Iterate the duplicated nodes and assign the next and random pointer for each nodes.
  3. Extract the duplicated nodes and recover the original list
  4. Although the time complexity and space complexity is O(n), the extra space complexity is O(1).
