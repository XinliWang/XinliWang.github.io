Merge k Sorted Lists

Aug 5, 2016 | Leetcode | Hits

Description:

Merge k sorted linked lists and return it as one sorted list. Analyze and describe its complexity.

Solution:

Analysis:

Solution1: Heap Sort. We can build a heap via Priority Queue. Each time we sort k nodes in the queue, then distract the least one and add a new one.
Solution2: Merge Sort. Time complexity is O(nlogn)
