This is the solution about Leetcode-Kth Largest Element in an Array.

Description

Find the kth largest element in an unsorted array. Note that it is the kth largest element in the sorted order, not the kth distinct element.

For example,
Given [3,2,1,5,6,4] and k = 2, return 5.

Note:
You may assume k is always valid, 1 ≤ k ≤ array’s length.

Solution

Analysis

Solution1

  1. sort array and get he kth largest element.
  2. Time O(nlogn), Space O(1)

Solution2

  1. Maintain a PriorityQueue with k size
  2. Put each element into PriorityQueue and get the peek at last.
  3. Time O(nlogk), Space O(k)

Solution3

  1. Turn to find the kth smallest number
  2. Take use of Quick Sort, each time find the position of pivot and get the length between left and pivot
  3. If the length equals to k, it’ll be the right one.
  4. if the length is less than k, loop the next round for right side of list.
  5. if the length is greater than k, loop the next round for left side of list
