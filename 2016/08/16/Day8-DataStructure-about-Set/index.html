##攻略模板：

  • 回忆数据结构：分析数据结构特性，比较类似数据结构差异
  • 实现数据结构
  • 实战训练使用该数据结构
  • 总结该数据结构特点，并总结算法题

###Java.util:

####自己回忆：

  • HashSet: only store value and in random order. It doesn’t allow to duplicate value in the hashset.
  • TreeSet: It’s ordered set.

####实现数据结构：
HashSet’s internal implementation is based on HashMap.

####学习后总结：

######HashSet vs. TreeSet

  • HashSet: 所有操作都是O(1),only allow unique element in HashSet. allowed null value. not thread-safe. If multiple threads access to the hashset, at least one thread modify set, it must be synchronized externally or use Collections.synchronizedSet() method. HashSet is faster than TreeSet.
  • TreeSet: 所有操作都是O(logn),duplicate-free collection of elements. not allowed null value. The default order is natural ordering. offers some handy method like first(),last().
  • LinkedHashSet: It provides insertion-ordered iteration.
