Longest Substring Without Repeating Characters

Aug 16, 2016 | Leetcode | Hits

This is the solution about Leetcode-Longest Substring Without Repeating Characters.

Description:

Given a string, find the length of the longest substring without repeating characters.

Examples:

Given "abcabcbb", the answer is "abc", which the length is 3.

Given "bbbbb", the answer is "b", with the length of 1.

Given "pwwkew", the answer is "wke", with the length of 3. Note that the answer must be a substring, "pwke" is a subsequence and not a substring.

Solution:

Analysis:

Solution1:

  1. Iterate through the string, store its character and index into HashMap.
  2. If there is a existed key, then record the size of hashmap which is the length of substringand compare to get the maximum one.
  3. remove all elements which are before this existed key(inclusive this element)
  4. store this new element and continue to the next one.
  5. At last return the maximum one from max and the size of hashmap.
  6. The time complexity is O(n^2)

Solution2:

  1. Iterate through the string, store its character and index into HashMap.
  2. Keep two pointers. Utilize right pointer to scan through string and left pointer move to the right of the same character last found if possible. Note that boty pointers only move forward.
  3. The length between two pointers is the result.
  4. The time complexity is O(n)

Solution3:

  1. Use Array instead of HashMap
  2. Initiate Array with -1 in order that left pointer move from the first location.
  3. Store location in Array based on the index of character.
